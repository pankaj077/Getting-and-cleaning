# Course project - getting and cleaning data

The directory has a run_analysis code which works if you have the necessary files in your system.
There is also a code book that describes the variables.
There is also a tidy.txt file.

Working of the run_analysis code-
1. Downloads the dataset, loads activity & features, Loads training and test sets.
2. Keep fields of mean or sd
3. Merging
4. Converts the `activity` and `subject` columns into factors
5. Create tidy text file
